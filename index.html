Lab 3: Apriori Algorithm

Problem 1: Grocery Store Example (Support Threshold s=2)

Given the following transactional database:

Transaction Database:
T1: {HotDogs, Buns, Ketchup}
T2: {HotDogs, Buns}
T3: {HotDogs, Coke, Chips}
T4: {Chips, Coke}
T5: {Chips, Ketchup}
T6: {HotDogs, Coke, Chips}

Task: Apply the Apriori algorithm with minimum support threshold s = 2 to find all frequent itemsets.

Problem 2: 5-Transaction Database (Minimum Support 60%)

Given the following transactional database:

Transaction Database:
T1: {A, B, D, E, F}
T2: {B, C, D, E, F, G}
T3: {A, D, E, H}
T4: {A, D, F, I, J}
T5: {B, D, E, K}

Task: Apply the Apriori algorithm with minimum support of 60% to find all frequent itemsets.
